# Counter App (React)

## 📌 Project Overview
The **Counter App** is a simple React application developed to understand the core concepts of React such as JSX, components, props, state, and the `useState` hook. The app allows users to increment, decrement, and reset a counter value, demonstrating dynamic state updates and reusable component design.

---

## 🧠 Topics Covered
- Create React App (CRA) setup  
- JSX  
- Functional Components  
- Props  
- State  
- `useState` Hook
