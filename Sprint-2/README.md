# Sprint 2 – React Basics (Create React App)

## 📌 Sprint Overview
Sprint 2 focuses on building a strong foundation in **React basics** using **Create React App (CRA)**.  
This sprint covers JSX, component-based architecture, props, state management, and the `useState` hook through hands-on mini projects.

---

## 🧠 Topics Covered
- Create React App (CRA) setup
- JSX syntax
- Functional Components
- Props
- State management
- `useState` Hook

---

## 🎯 Learning Outcomes
By the end of this sprint, you will be able to:
- Create and reuse React components
- Manage and update component state
- Pass data and functions using props
- Build interactive user interfaces using React hooks

---

## ✅ Tasks Implemented

### 1️⃣ Counter App
A simple counter application demonstrating:
- State initialization using `useState`
- Increment, decrement, and reset functionality
- Dynamic UI updates based on state changes

---

### 2️⃣ Reusable Card / Button Components
This task focuses on **component reusability**:
- Custom reusable Card component
- Reusable Button component with props
- Styling via CSS
- Props used to customize content and behavior

---

### 3️⃣ Feedback Form with Comment Display
An interactive form application that:
- Accepts user feedback through an input form
- Stores comments using React state
- Displays submitted feedback dynamically
- Demonstrates controlled components

---

## 🛠️ Technologies Used
- React (Create React App)
- JavaScript (ES6)
- JSX
- CSS
